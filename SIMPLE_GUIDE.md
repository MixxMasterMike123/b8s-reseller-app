# B8Shield Simple Guide

## What is B8Shield?

B8Shield is a Swedish reseller portal for fishing lure protection products. It has two main parts:

- **B2B Portal**: For resellers to manage customers and orders
- **B2C Shop**: For customers to buy products directly

## Quick Links

- **B2B Portal**: https://partner.b8shield.com
- **B2C Shop**: https://shop.b8shield.com
- **Admin Login**: micke.ohlen@gmail.com

## Main Features

### For Resellers (B2B)
- Customer management
- Product catalog
- Order processing
- Marketing materials

### For Customers (B2C)
- Product browsing
- Shopping cart
- Checkout process
- Account management

## Technology

- **Frontend**: React + Vite
- **Styling**: Tailwind CSS
- **Backend**: Firebase
- **Database**: Firestore
- **Hosting**: Firebase Hosting

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to Firebase
firebase deploy
```

## Key Files

- `src/App.jsx` - Main app with routing
- `src/contexts/` - State management
- `src/pages/` - Main pages
- `src/components/` - Reusable components
